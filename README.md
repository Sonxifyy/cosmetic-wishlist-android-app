# Cosmetic Wishlist Android App

Mobile Android application developed in Java that allows users to create and manage a personal cosmetic wishlist.

## Features
- User authentication (Google / Firebase login)
- Integration with external API
- Firebase Firestore database (CRUD operations)
- Realtime database integration
- Firebase Storage for media files
- Multiple activities and navigation
- RecyclerView lists with item click functionality
- Dynamic UI updates from database data

## Technologies
- Java
- Android Studio
- Firebase Firestore
- Firebase Authentication
- RecyclerView
- REST API integration
- Android SDK

## Description
The application allows users to browse cosmetic items, add them to a wishlist and manage their personal collection. Data is stored in Firebase databases and displayed dynamically using RecyclerView components.
